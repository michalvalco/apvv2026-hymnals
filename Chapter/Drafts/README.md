# Chapter Drafts

This folder will contain chapter draft files (and their associated audit documents) as drafting begins.

## Versioning convention

- `Chapter_Draft_v0.1_[scope].md` — initial drafts, scoped (e.g., `Chapter_Draft_v0.1_Locus1.md` for the §4.1 Eucharist locus draft)
- `Chapter_Draft_v0.2_[scope].md` — second-pass drafts after audit fixes
- `Chapter_Draft_v1.0_Full.md` — integrated full chapter draft (post-Phase-4)
- `Chapter_Bibliography.md` — bibliography (assembled separately, integrated at finalisation)
- `Audit_[scope].md` — audit documents (e.g., `Audit_Locus1_Eucharistia.md`)

## Drafting sequence

Per `../Chapter_Master_Plan.md`, the drafting sequence is:

1. §4.1 (Locus 1 — Eucharistia) — first, because the source base is most developed and the Theologos paper provides scaffolding
2. §4.2 (Locus 2 — Ekleziológia) — second, because the compass artifact provides scaffolding for §3 + §4.2
3. §4.3 (Locus 3 — Soteriológia)
4. §4.4 (Locus 4 — Písmo a tradícia)
5. §3 (Dogmatické profily) — drafted after the *loci* analyses, to ensure the profiles match what the comparative work produced
6. §5 (Transfer a recepcia)
7. §1 (Úvod) and §6 (Záver) — drafted last, to reflect the actual work the chapter does

§2 (Konfesionálny kontext) can be drafted at any point; it's largely independent of the comparative work in §§3–5.
